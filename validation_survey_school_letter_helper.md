# 検証用アンケート設計書：School Letter Helper

本資料は、外国人保護者や支援者に「School Letter Helper」のMVP（テスト版）を利用してもらった後に実施する、仮説検証用のアンケート項目（Googleフォーム等での構築用）の設計書である。

---

## 1. アンケート設計の目的

MVP設計（`business_ideas_jvb.md`）の成功条件およびKPIを測定し、以下の仮説を検証する。
1. **課題の強さ**: 学校文書の理解・返信にどの程度の痛みを感じているか。
2. **ソリューションの受容性**: 今回提供した機能（要約・ToDo・返信文生成）が課題解決に役立っているか。
3. **継続利用意向**: 今後も使い続けたいと思うか（7日後継続率や再利用意向）。
4. **推薦意向**: 他の外国人保護者や支援団体に紹介したいと思うか。

## 3. KPI判定マニュアル（MVP成功条件との連動）

アンケート回答結果から、以下の成功条件が達成されたかを判定する。

| 測定する成功条件 (business_ideas_jvb.md より) | 対応するアンケート項目 | 達成目標 |
| :--- | :--- | :--- |
| **10人以上が実際のおたよりを試す** | Q5で「はい、自分のおたよりで試した」の回答数 | **10件以上** |
| **うち5人以上が「また使いたい」と回答** | Q5が「はい」の回答者のうち、Q6で「ぜひ使い続けたい」と答えた数 | **5件以上** |
| **3人以上が返信文生成またはToDo機能を使う** | Q4-3 または Q4-4 で「4（役に立った）」以上を選択した数 | **3件以上** |
| **支援者側から「学校・団体で使える可能性がある」と評価される** | Q1で「ボランティア」「NPO」等を選択した回答者から、Q8等で肯定的なFBを得る | **1件以上** |
