# School Letter Helper LP

Cloudflare Pagesで公開するための最小構成の静的LPです。

## Files

- `index.html`: トップページ
- `style.css`: 共通スタイル
- `privacy.html`: プライバシーポリシー

## Before Deploy

次のプレースホルダーを差し替えてください。

- `index.html`: `https://forms.gle/REPLACE_WITH_YOUR_FORM_ID`
- `privacy.html`: `REPLACE_WITH_YOUR_EMAIL`

## Cloudflare Pages

HTML直書き構成なので、Cloudflare Pagesの設定は次で公開できます。

- Framework preset: `None`
- Build command: 空欄
- Build output directory: `/`

## Local Preview

`index.html` をブラウザで開くだけで確認できます。
