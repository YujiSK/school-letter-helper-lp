# テストユーザー向け利用ガイド：School Letter Helper

本ドキュメントは、「School Letter Helper」のテスト版（モックプロトタイプ）をご体験いただく参加者に向けた案内資料です。
このガイドのテキストをコピーし、URL部分を実際のURLに書き換えてそのまま参加者へお送りいただけます。

---

## 1. テスト概要 (Test Overview)

*   **テストの目的**: 
    日本の学校のプリント（おたより）をスマホで撮影するだけで、「やさしい日本語」や「母語」に要約・ToDo化し、先生宛ての返信メールを作る機能が、実際に外国人保護者や支援員の皆様の役に立つかどうかを検証すること。
*   **所要時間の目安**: 
    約 3〜5 分程度
*   **必要なもの**: 
    インターネットに接続されたスマートフォンまたはPC、テスト用のサンプル画像（以下からダウンロード可能）
